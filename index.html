CLMinesweeper

This project is a commandline version of the famous office time killer Minesweeper. To play just pass in the board width board height and number of mines as command line argeuments.

Installation

  1. Clone the repository
  2. Compile with g++ -std=c++11 -o minesweeper minesweeper.cpp
  3. Run with ./tweetGen w h mines where w is an int for witdth h is a int for height and mines is an int for number of mines
